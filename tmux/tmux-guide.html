tmux Field Guide.

A practical reference for windows, panes, and sessions — with a quiz to test yourself.

Almost every tmux command starts with the prefix: Ctrl+b by default. Throughout this guide, then means "release the prefix, then press." So Ctrl+b then % means tap the prefix, let go, then hit %.

The mental model

A session holds one or more windows (like browser tabs). Each window is split into panes (the rectangles you actually type in).

Sessions

tmuxStart a new session (from the shell).
tmux new -s workStart a new named session called "work".
tmux attach -t workRe-attach to the "work" session.
Ctrl+bthendDetach — leave the session running in the background.
tmux lsList all running sessions.

Windows (tabs)

Ctrl+bthencCreate a new window.
Ctrl+bthennNext window.
Ctrl+bthenpPrevious window.
Ctrl+bthen0…9Jump straight to window by number.
Ctrl+bthenwVisual list of windows to pick from.
Ctrl+bthen,Rename the current window.

Splitting into panes

Ctrl+bthen%Split vertically — new pane to the right.
Ctrl+bthen"Split horizontally — new pane below.
Ctrl+bthen←/→/↑/↓Move focus between panes.
Ctrl+bthenxClose the current pane (asks to confirm).
Memory hook: % looks like a tall line dividing left/right (vertical split). " sits up high, hinting a top/bottom (horizontal) split.

Resizing panes

Ctrl+bthen holdCtrl+←/→/↑/↓Resize the active pane one cell at a time.
Ctrl+bthenzZoom — toggle the active pane to fullscreen and back. Hugely useful.

Changing the layout

tmux ships five preset layouts. Cycle them, or jump to one directly.

Ctrl+bthenSpaceCycle through the next preset layout. Press repeatedly to flip a vertical split into a horizontal one and beyond.
Ctrl+bthenAlt+1even-horizontal — all panes side by side.
Ctrl+bthenAlt+2even-vertical — all panes stacked top to bottom.
Ctrl+bthenAlt+3main-horizontal — one big pane on top, the rest in a row below.
Ctrl+bthenAlt+4main-vertical — one big pane on the left, the rest stacked right.
Ctrl+bthenAlt+5tiled — an even grid.
Want to flip a vertical split into a horizontal one? Press Ctrl+b then Space to cycle, or jump directly with Alt+2 (even-vertical / stacked) vs Alt+1 (even-horizontal / side by side).

Rearranging panes

Ctrl+bthen{Swap the active pane with the one before it.
Ctrl+bthen}Swap the active pane with the one after it.
Ctrl+bthenoRotate focus to the next pane.
Ctrl+bthenqBriefly show pane numbers; press a number to jump to it.
Ctrl+bthen!Break the active pane out into its own new window.

Scroll & copy mode

Ctrl+bthen[Enter copy/scroll mode. Use arrows or PgUp/PgDn to scroll back through history. Press q to exit.
Ctrl+bthen]Paste the most recently copied buffer.

Moving a pane into the current window (join-pane)

This is the answer to "take a pane from window 3 and dock it as a split in my current window." It is done from the command prompt.

Ctrl+bthen:Open the tmux command prompt.
join-pane -s 3Pull the pane from window 3 into the current window as a new split.
join-pane -s 3.1 -hPull pane 1 of window 3 in as a horizontal neighbour (-h = left/right, -v = top/bottom).
break-paneThe inverse — eject the current pane back out into its own window.
move-pane -t 2Send the current pane into window 2.
Why it feels odd: there is no default key binding for join-pane — you type it at the : prompt. Many people bind it in ~/.tmux.conf, e.g. bind J command-prompt -p "join from:" "join-pane -s '%%'".

Moving / merging windows & sessions

swap-window -s 2 -t 1Swap the positions of window 2 and window 1.
move-window -t 5Renumber the current window to position 5.
move-window -s other:1 -t 0Pull window 1 from session "other" into this session at slot 0.
Ctrl+bthen.Move the current window to a different index (prompts for the number).

Synchronize & broadcast

setw synchronize-panes onType once, send to every pane in the window at the same time. Set off to stop.

Config & reload

source-file ~/.tmux.confReload your config without restarting tmux (run from the : prompt).
set -g mouse onEnable mouse: click panes to focus, drag borders to resize, scroll to enter copy mode.
set -g prefix C-aRemap the prefix from Ctrl+b to Ctrl+a (a common preference).
Pro move: tmux kill-session -t work tears down a session by name; tmux kill-server nukes everything. Detaching (Ctrl+b then d) is almost always what you actually want instead.
